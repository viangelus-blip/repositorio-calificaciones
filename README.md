# Repositorio de Calificaciones (Programación Estructurada)

Programa en Python que permite registrar materias y calificaciones (0 a 10), calcular promedio, identificar aprobadas/reprobadas (umbral 5.0), y mostrar la nota más alta y más baja, con validación básica de entradas.

## Requisitos
- Python 3.10+ (recomendado)

## Ejecución
1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta:

```bash
python calificaciones.py
